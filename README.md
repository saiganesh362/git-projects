# Library Management System (Python OOP)

## 📖 Overview
A modular Library Management System built using Object-Oriented Programming concepts in Python.  
It manages books, members, borrowing/returning, file persistence, and provides a simple CLI interface.

## 🗂️ Project Structure
- `book class.py` → Book class
- `member class.py` → Member class
- `library class.py` → Library class
- `core functionality.py` → Borrow/return/search/statistics
- `file operations.py` → JSON save/load
- `UI.py` → Menu-driven CLI

## ⚡ Features
- Add/remove books
- Register members
- Borrow/return books with limits
- Search by title, author, ISBN
- View statistics
- Save/load data with JSON
